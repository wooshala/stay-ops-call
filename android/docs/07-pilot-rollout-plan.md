# 07. Pilot Rollout Plan

## 파일럿 대상 기기 정의

- 제조사/OS: Samsung One UI + Android 13/14
- 모델군: Galaxy A 시리즈 중 1종(운영 확정 모델)
- 확장 정책: 파일럿 성공 기기군부터 단계적 확대

## 비지원 기기 안내 정책

- 앱 온보딩 시 지원기기 정책 표시
- 비지원 OS/제조사 조합은 "제한 지원"으로 명시
- 운영 매뉴얼에 제조사별 통화녹음/파일 접근 차이 공지

## 기기별 리스크 목록

1. 제조사별 통화녹음 저장구조 상이
2. SAF 권한 해제/초기화로 폴더 접근 손실
3. 백그라운드 제한(절전/최적화)으로 워커 지연
4. 파일명 규칙 변경으로 중복판정 실패 가능
5. 네트워크 불안정 시 업로드 누락/지연

## 대응책

1. SAF 사용자 지정 폴더를 단일 기준으로 채택
2. 앱 시작 시 권한/폴더 무결성 점검 및 재선택 안내
3. WorkManager + 재시도 백오프 + 상태 가시화 제공
4. 파일 해시(sha256) 고정 기반 중복 제어
5. 실패 알림 + 재업로드 버튼 + 운영 모니터링 대시보드

## 운영 단계

- 0주: 내부 QA(기기 1종, 샘플 통화 세트)
- 1~2주: 파일럿 매장 소규모 도입
- 3주: 실패 유형 분석/보정
- 4주: 모델 확장 여부 결정
