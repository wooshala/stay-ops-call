# 01. Product Overview

## 목적

업장폰에서 통화 종료 후 생성된 녹음 파일을 자동(또는 반자동) 수집하여 서버로 업로드하고, STT/분석/검수/CRM 처리 상태를 모바일에서 안정적으로 추적한다.

## MVP 범위

- 로그인 + 기기 등록
- SAF 기반 녹음 폴더 지정 및 스캔
- 중복 없는 업로드 큐
- 업로드 성공/실패/재시도 관리
- 서버 처리 상태 조회
- 최근 통화/결과 목록, 검수 필요/실패 확인, 재업로드

## 지원 기기 정책

- 초기 지원: **Samsung One UI + Android 13/14**
- 파일럿 모델: **Galaxy A 시리즈 1종(운영 협의로 확정)**
- 전략: 범용 최적화가 아닌 지정 환경 100% 안정성 우선

## 비지원 기기 안내

- One UI 외 제조사 커스텀 ROM
- Android 12 이하 또는 파일 접근 정책이 다른 구형 단말
- 통화녹음 파일 구조가 폐쇄적이며 SAF 접근이 불안정한 기기

## 단계 전략

- 1차: SAF 기반 수동 폴더 지정 + 안정적 백그라운드 동기화
- 2차: 삼성 기본 통화녹음 경로 자동 탐지 보조 기능
- 3차: 기기군 확장(검증 통과 모델만 순차 지원)
